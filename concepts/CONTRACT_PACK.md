## 1. Определение "Работа ученика" и связь с уроками ✅ РАЗРЕШЕНО

**Проблема:**
- `homework.md:7`: "Результат прохождения урока с ревью"
- `lesson.md:38-43`: Существуют категории "Урок с ревью" и "Урок без ревью"
- `lesson.md:44-46`: "Урок с отставанием" упоминает "работу на ревью"

**Решение принято:** Работа ученика учитывается только для уроков с ревью.

**Внесенные изменения:**
1. `homework.md:7`: Уточнено определение - "Результат прохождения урока с ревью. Самостоятельная работа, которую ученик готовит по результатам урока с ревью."
2. `homework.md:36`: Добавлена рамка - "Существует только для уроков с ревью"
3. `homework.md`: Добавлен антипример - "для урока без ревью работа ученика не создается и не отслеживается"
4. `lesson.md:44`: Уточнена категория - "Урок с ревью, по которому ученик опаздывает подготовить работу"

## 2. Категории Курса: пересечение и взаимосвязи ✅ РАЗРЕШЕНО

**Проблема:**
- `course.md:21-47`: Категории пересекаются без четких правил
- Курс может быть одновременно: Активным, Оплаченным, В расписании

**Решение принято:** Определить иерархию категорий:
1. **Жизненный цикл курса:** Разрабатываемый → Активный → Архивный
2. **Атрибуты активного курса:** Оплаченный, В расписании (только для активных курсов)
3. **Состояния прогресса ученика:** Начатый, Завершенный (только для активных курсов)

**Внесенные изменения:**
1. `course.md`: Реструктурированы категории с явным разделением на группы
2. Уточнено, что "Оплаченный" и "Курс в расписании" могут быть только для активных курсов
3. Уточнено, что "Начатый" и "Завершенный" относятся к активным курсам
